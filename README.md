# 🏨 WanderLust – Full Stack Hotel Booking Platform

WanderLust is a full-stack web application inspired by Airbnb that allows users to explore, list, and book hotels or travel accommodations.  
The platform provides authentication, property listings, booking management, and review functionality.

---

## 🚀 Features

- User authentication and authorization
- Create, edit, and delete hotel listings
- Upload images for properties
- View hotel listings with detailed information
- Add and manage reviews for listings
- Secure login and registration system
- Responsive UI for better user experience

---

## 🛠️ Tech Stack

### Frontend
- HTML
- CSS
- Bootstrap
- EJS (Embedded JavaScript Templates)

### Backend
- Node.js
- Express.js

### Database
- MongoDB
- Mongoose

### Authentication & Security
- Passport.js
- Express-session
- Flash messages

### Architecture
- MVC (Model View Controller)

---

## 📂 Project Structure
wanderlust-hotel-booking
│
├── controllers # Handles application logic
├── models # Database schemas
├── routes # Application routes
├── views # EJS templates
├── public # Static files (CSS, JS, images)
├── utils # Utility functions
└── app.js # Main server file

## 📦 Installation

### 1 Clone the repository
git clone https://github.com/YOUR_USERNAME/wanderlust-hotel-booking.git

### 2 Navigate to project folder
cd wanderlust-hotel-booking

### 3 Install dependencies
npm install

### 4 Start the server
node app.js

The application will run at:
http://localhost:3000
